# Block 005D0000 — Function Documentation
**Functions documented**: 370
**State-mutating functions**: 0 (none — this block is entirely SMEDS multimedia framework)

## Block Overview
This block contains the **SMEDS (Sid Meier's Entertainment Development System) multimedia framework** — the proprietary middleware library used by Civ2 for sprite rendering, audio playback (WAV mixing, CD audio, MIDI), timer management, file I/O, window/control management, palette handling, debug logging, GIF loading, and RLE encoding/decoding. All globals are in the 0x00637Exx-0x00638Bxx (framework config) and 0x006E47xx-0x006E50xx (framework object instances) ranges. **No game state addresses (0x0063-0x006C game state ranges) are written.**
