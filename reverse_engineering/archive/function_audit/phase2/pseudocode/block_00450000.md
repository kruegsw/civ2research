**Address**: 0x00453DA0 | **Size**: 120 bytes
**Category**: GL.wonder | **Porting**: P

**Parameters**: (wonderId: int) → bool

Checks whether a wonder has been made obsolete by any civ researching its obsolescence advance.

```
function is_wonder_obsolete(wonderId):
    obsoleteTechId = WONDER_OBSOLETE_TECH[wonderId]   // DAT_0064ba28[wonderId], signed byte
    if obsoleteTechId < 0:
        return false           // no obsolescence tech defined

    for civ = 1 to 7:
        if civ_has_tech(civ, obsoleteTechId):
            return true        // someone researched the obsoleting advance

    return false
```

### FUN_00453e51 — `civ_has_active_wonder`

**Address**: 0x00453E51 | **Size**: 142 bytes
**Category**: GL.wonder | **Porting**: P

**Parameters**: (civId: char, wonderId: int) → bool

Returns true if civId owns an active (non-obsolete) wonder. Special case: wonderId 0x14 (Statue of Liberty) is always false if scenario flags bit 0x80 and scenario bit 0x01 both set.

```
function civ_has_active_wonder(civId, wonderId):
    // Special scenario override for Statue of Liberty
    if wonderId == 0x14 AND (scenarioFlags & 0x80) != 0 AND (scenarioBits & 1) != 0:
        return false

    cityIndex = get_wonder_city(wonderId)
    if cityIndex < 0:
        return false

    return CITY_DATA[cityIndex].owner == civId     // DAT_0064f348[cityIndex*0x58]
```

### FUN_00456f8b — `calc_patience_threshold`

**Address**: 0x00456F8B | **Size**: 211 bytes
**Category**: GL.diplomacy | **Porting**: R

**Parameters**: (civA: int, civB: int) → int

Computes the maximum patience value for civB toward civA, based on the current live attitude score and treaty flags.

```
function calc_patience_threshold(civA, civB):
    threshold = 2

    if liveAttitudeScore < 25:                           // DAT_0064b114
        threshold = 3
    if liveAttitudeScore > 60:
        threshold -= 1

    // Statue of Liberty bonus
    if civ_has_active_wonder(civA, STATUE_OF_LIBERTY):   // wonder 0x14
        threshold += 1

    // Embassy bonus
    if treaty[civA][civB] & 0x04:                        // embassy flag
        threshold += 1

    // Alliance bonus
    if treaty[civA][civB] & 0x08:                        // alliance flag
        threshold += 2

    // Hatred override
    if treaty[civA][civB].byte1 & 0x20:                  // hatred/vendetta bit
        threshold = 2

    return threshold
```

**Calls**: civ_has_active_wonder (FUN_00453e51)

### FUN_0045705e — `ai_evaluate_diplomacy`

**Address**: 0x0045705E | **Size**: 6616 bytes (xlarge)
**Category**: AI.diplomacy | **Porting**: PA

**Parameters**: (civA: int, civB: int)

The **master diplomacy evaluation function**. Computes all AI negotiation variables: tribute amount, tech demand, attitude score, war desire, third-party targets, and alliance/ceasefire willingness. Sets ~20 global DAT_ variables used by the subsequent negotiation logic.

```
function ai_evaluate_diplomacy(civA, civB):
    // ── Nuclear Capability ──
    civB_has_nukes = check_nuclear_capability(civB)         // FUN_00598d45
    allianceCount = 0
    for civ = 1 to 7:
        if treaty[civA][civ].byte1 & 0x20:                 // hatred/allied
            allianceCount++
        if civ is AI and alive and difficulty > 3:
            civB_has_nukes = 1                              // force nuke awareness

    // ── Relative Power Assessment ──
    civB_is_weaker = (difficulty[civA] < difficulty[civB])   // DAT_00655c22
    DAT_0064b128 = 0  // wants_map flag

    // ── Record last tribute paid (if in AI-initiated mode) ──
    if DAT_00626a20 != 0 AND DAT_00626a1c == 0:
        DAT_0064b108 = lastTributeTurn[civA][civB]

    // ── Initialize output variables ──
    tributeAmount = 0                                       // DAT_0064b0ec
    techDemand = 0                                          // DAT_0064b118
    wantsMore = 0                                           // DAT_0064b0f8
    wantsNothing = 1                                        // DAT_0064b148

    // ══ TECH VALUE SCAN ══
    // For each continent (1..63):
    for continent = 1 to 63:
        if civA has units on continent with count > 1:
            // Compare civA's and civB's military strength on this continent
            if civA_strength[continent] > civB_strength[continent]:
                // civB has exploitable tech: accumulate tribute
                tributeAmount += (civA_strength * difficulty_factor) /
                                 (civB_strength + 1)
            else:
                // Check mutual tech presence for map exchange
                if both have troops here: wantsMap = 1

                // Compute tech differential score
                if civA has no cities on continent:
                    if civB has no cities either:
                        techDemand += (civB_strength - civA_strength) / 4
                    else:
                        techDemand += (civB_total - civA_strength) / 2
                else:
                    // civA has cities: if continent count < threshold
                    if allianceCount * -50 + 200 >= currentTurn OR
                       difficulty[civA] > 6 OR difficulty[civB] > 6:
                        techDemand += civB_strength
                    else:
                        techDemand += civB_strength - civA_strength

    // ── Great Library bonus ──
    if civ_has_active_wonder(civB, GREAT_LIBRARY):          // wonder 9
        techDemand += techDemand / 4

    // ── Should Declare War Check ──
    shouldDeclareWar = 0                                     // DAT_0064b11c
    if difficulty[civA] == 7 AND cities[civA] > 4
       AND cities[civB] > 1 AND government[civA] != anarchy
       AND difficulty > 0 AND currentTurn > 200:
        shouldDeclareWar = 1

    // ── Attitude Score ──
    attitudeScore = civ[civB].attitude[civA]                // DAT_0064b114
    // Adjust for: shouldDeclareWar, provocation flag (0x10),
    //             military ratio, anarchy state

    if shouldDeclareWar OR (treaty has 0x10 AND military_ratio < 3:1):
        techDemand = clamp(techDemand, difficulty[civB]*2, 9999)
        if no mutual tech: techDemand = clamp(0, treasury/50/(8-diff))
        wantsMore = 1
        tributeAmount = 0
        if attitudeScore < 76: attitudeScore = 75

    // ── Nuclear stance adjustments ──
    if civB_has_nukes AND civB_is_weaker:
        wantsMore = 1
    if civB is in anarchy AND no alliance:
        wantsMore = 1; techDemand = clamp(100, 9999)

    // ── Reputation adjustments ──
    if civA has nuclear weapons AND civB has no alliance:
        set treaty bit 0x100 (nuke awareness)
        wantsMore = 1

    // ── Government personality modifier ──
    // Divide tributeAmount by (patience-1)/2 + 1
    tributeAmount /= ((patience[civA] - 1) / 2 + 1)

    // ── Alliance flag adjustments ──
    if treaty has alliance (0x08):
        attitudeScore -= 25
    if treaty.byte1 has vendetta (0x20):
        attitudeScore += 25

    // ── Personality/treaty violation score ──
    if civA has patience byte != 0:
        attitudeScore += (patience - 1 - treatyViolations[civB][civA]) * 5

    // ── Attitude thresholds ──
    if attitudeScore < 26:
        techDemand /= 2; wantsMore = 0
    elif attitudeToLevel < 4:
        techDemand = techDemand * 2 / 3
    elif attitudeScore > 74:
        techDemand = techDemand * 3 / 2; wantsMore = 1

    // ── Attacked flag ──
    if treaty.byte1 & 0x08: wantsMore = 1

    // ── Scenario special rules ──
    if scenario & 0x80 AND scenarioBits & 0x8000:
        // hardcoded civ pairs: (6,7), (3,6), (3,1) force war/peace

    // ── Great Wall / UN wonders ──
    if civ_has_active_wonder(civA, GREAT_WALL) OR
       civ_has_active_wonder(civA, UNITED_NATIONS):
        shouldDeclareWar = 0
        DAT_0064b128 = wantsMore  // save for later
        wantsMore = 0
        attitudeScore -= 10
        clear_treaty_flags(civB, civA, 0x10000)

    // ── Scale by patience ──
    if civA.patience != 0:
        techDemand += patience * techDemand / 2

    // ── Treaty-level multipliers ──
    if treaty has provocation (0x10):
        techDemand *= 2
    elif treaty.byte2 & 0x01:
        techDemand /= 4

    // ── Unit proximity check (border violations) ──
    violationCount = 0
    for each unit of civA:
        if unit is near civB territory AND border distance < 3:
            techDemand *= 2
            violationCount++

    // ── Compute final tribute ──
    tribute = ((difficulty + 1) * techDemand + 16) / 32 * 50
    // Clamp to civA's treasury
    if tribute > treasury AND tribute < treasury * 2 AND treasury > 49:
        tribute = (treasury / 50) * 50
    if tribute == 0 OR civB_military * 3 < civA_military:
        wantsMore = 0
    // Reset if 10+ consecutive demands
    if consecutiveDemands[civB][civA] > 9:
        consecutiveDemands = 0; wantsMore = 0

    // ── Final want/don't want ──
    if wantsMore == 0:
        if treaty has alliance: wantsNothing = 0
    else:
        wantsNothing = 2     // aggressive demand mode

    wantsDeclareWar = DAT_0064b13c = wantsMore

    if treaty & 0x10: shouldDeclareWarScore -= 2
    if scenario & 0x80 AND anarchy: shouldDeclareWarScore -= 1
    if patience > 0: shouldDeclareWarScore -= patience / 2

    // ── Third-party target selection ──
    thirdPartyCivForCancel = -1                            // DAT_0064b100
    thirdPartyCivForTechDemand = -1                        // DAT_0064b0fc
    thirdPartyCivForWar = -1                               // DAT_0064b104
    // Scan all civs: find allies of civA at war with civB,
    // find weakest ally of civB, find best war target

    // ── War weariness check ──
    if allyCount == 0: clear_treaty_flags(civB, civA, 0x100000)
    shouldDeclareWarScore -= government_aggression[civB]
    if civA_military < civB_military: shouldDeclareWarScore -= 1

    // ── Should declare war final check ──
    if shouldDeclareWarScore > (isFriendly ? 1 : 0)
       AND shouldDeclareWar == 0
       AND no alliance AND no ceasefire
       AND random check passes:
        techDemand = 0  // will provoke instead

    // ── Caravan/trade violation scan ──
    borderViolations = 0                                    // DAT_0064b0f4
    for each city belonging to civB near civA border:
        borderViolations++

    // ── Should initiate diplomacy ──
    wantsToInitiate = 0                                     // DAT_0064b0e8
    // Complex multi-condition: if enough violations, enough attitude,
    //   enough military superiority, enough tech advantage → wantsToInitiate = 1
    if wantsToInitiate:
        wantsMore = 0; wantsNothing = 0
        if attitudeScore > 29: attitudeScore = 30

    // ── War weariness (Great Wall/UN) ──
    if DAT_00626a20 AND patience < threshold
       AND military > 4 AND wantsMore
       AND no provocation/ceasefire flags
       AND not scenario override:
        if civB_military * 2 < civA_military * 3 AND no shouldDeclareWar:
            wantsMore = 0
```

```
        bestGiveValue = bestReceiveValue = secondReceiveValue = 0
        bestGiveTech = bestReceiveTech = secondReceiveTech = -1
        wonderBlockingTech = -1

                if bHas AND not aHas:
                    countB++

                    check_wonder_blocks(techId)
                    // Track best tech to receive
                    value = tech_value(civA, techId)
                    if value >= bestReceiveValue:
                        bestGiveTech = techId; bestReceiveValue = value

                if aHas AND not bHas:
                    // Track best tech to give away
                    value = tech_value(civB, techId)
                    if value > bestGiveValue:
                        secondReceiveTech = bestReceiveTech
                        bestReceiveTech = techId
                        bestGiveValue = value

        if scanOnly: return

        // ── Pre-exchange checks ──
        diplo_show_attitude_header(civA, civB)

        if not isAllied:
            // Check: at war, provocation, or trade violations → refuse
            if treaty & 0x10 OR treaty.byte1 & 0x10 OR borderViolations > 0:
                if mode == 0: show_mood(3/4); show("NOEXCHANGEMEDIUM/MAD")

               OR attitudeLevel > 5:
                if mode == 0: show_mood(3/4); show("NOEXCHANGE...")
                return

        if wonderBlockingTech >= 0 AND difficulty > 3:
            if mode != 0: return
            show_wonder_name; show("NOEXCHANGEWONDER")
            return

        // ── Tech sell attempt (if they have more techs) ──
        if countA < countB:
            if attitudeLevel > threshold AND not allied:
                result = diplo_sell_tech(civA, civB, mode)
                if result: return
                if mode != 0: return
                show("NOEXCHANGEMEDIUM")
                return

           AND attitudeLevel > threshold

            result = diplo_sell_tech(...)

            if mode == 0: show("NOEXCHANGENOW")
            return

        if bestReceiveTech < 0 AND bestGiveTech >= 0 AND isAtWar
           AND tech_level criteria met AND patience < 2:
            grant_tech(civA, bestGiveTech, civB, 0, 0)

            bestGiveTech = -1
            if mode > 1: return
            civ[civB].patience += 2
            return

        // ── No exchange possible ──
        if bestGiveTech < 0 OR bestReceiveTech < 0:
            result = diplo_sell_tech(...)
            if mode == 0: show("NOEXCHANGENOW")
            return

        if mode > 1: return

        // ── Show emissary dialog ──
        result = diplo_ai_emissary(civA, civB, tileX, tileY)
        if result == 0: return

        // ── Present exchange offer ──
        set_sprite(1, bestGiveTech_name)

        set_sprite(3, bestReceiveTech_name)
        if secondReceiveTech >= 0:

        result = show_dialog(0)

            if various_checks_fail:

        grant_tech(civB, bestReceiveTech, civA, 0, 0)

        update trackers
        adjust_attitude(civB, civA, -techValue * 2)
        set civ flags 0x80 (diplomacy occurred)
        wantsMore = 0
        if mode == 0: return     // one exchange per meeting
```

Converts a gold payment to attitude adjustment points using a diminishing-returns formula.

```
function calc_gold_to_attitude(goldAmount):
    attitude = 0
    divisor = 10
    bracket = 50

    while goldAmount > 0:
        portion = clamp(goldAmount, 0, bracket)
        attitude += portion / divisor
        divisor += 5
        goldAmount -= bracket
        bracket = 100

    return attitude
```

```
                if much weaker: adjustAmount = 0

        adjust_attitude(civB, civA, 50)
        ai_evaluate_diplomacy(civA, civB)
        show "CANCELALLY" dialog
        patience[civA]++
        break_alliance(civA, civB)

    return result
```
