### Porting Notes
**Partially ported** to `engine/combat.js:resolveCombat()`. The JS port covers:
- Base attack/defense calculation with terrain, veteran, fortification modifiers
- Pikeman double defense vs mounted
- Aegis defense bonus vs air/missiles
- Air vs unarmed ships (halved defense, FP=1)
- Air vs land (both FP=1)
- Helicopter vs submarines (x8 attack)
- Fractional movement penalty
- Difficulty scaling for barbarian attacks
- Round-by-round combat loop with PRNG

**MISSING from JS port**:
1. **Great Wall wonder double-roll** — binary does a second random roll when attacker wins; if defender wins the re-roll, outcome is reversed. JS port does not implement this.
2. **Diplomatic consequences** — treaty breaking, sneak attack notifications (SNEAK, BREAKCEASE, CANCELPEACE), alliance notifications (ALLYUNDERATTACK, ALLYATTACKING). Reducer.js handles first-contact ceasefire but not the full treaty-break cascade.
3. **City capture logic** — the binary reduces city size on capture, destroys size-0 cities, reassigns production, handles civ elimination. Reducer.js has partial capture but not the full binary fidelity.
4. **SDI/SAM/Coastal Fortress building bonuses** — JS port has SAM(27), Coastal(28), SDI(17) in `combat.js`. These appear correct.
5. **Ransom mechanic** — killing a high-value barbarian unit awards gold (difficulty * 50).
6. **Multiple unit combat messages** — DAT_006acb0c tracking consecutive kills.
7. **Nuke/cruise missile special resolution** — binary delegates to FUN_0057f9e3 for nuclear weapons; JS has NUKE action in reducer but different path.
8. **Submarine detection / retreat logic** — defender withdrawal from combat.
9. **Amphibious attack** — sea unit attacking land gets doubled FP.
10. **Combat sound mapping** — 40+ sound IDs mapped by unit type/domain.

    if NOT has_horseback_riding:
        if rand() % 3 == 0: base_unit = 0x10  // Phalanx
        else: base_unit = 0x0f  // Legion
        if has_iron_working: base_unit = 0x11
    if has_gunpowder: base_unit = 0x12
    if has_conscription: base_unit = 0x14
    if has_metallurgy: base_unit = 0x0b

    // Secondary unit type
    secondary = 8 (Catapult era)
    if has_magnetism: secondary = 7
    if has_chemistry: secondary = 5
    if has_invention: secondary = 4

    // 50% chance of primary or secondary
    chosen = (rand() % 2 == 0) ? base_unit : secondary

    // Create unit at hut tile
    spawn_unit(chosen, owner, x, y)
    // Set no home city
    unit.home_city = 0xFF

case 2 (GOLD):
    gold_amount = 50
    if rand() % 3 == 0:
        sub_roll = rand() % 10 - difficulty + 2
        if sub_roll < 5: gold_amount = 25
        else: gold_amount = 100
    if turn > 1000: gold_amount *= 2

    civ[owner].gold += gold_amount
    show "SURPRISEMETALS"

case 3 (BARBARIANS):
    // Suppress if early game or no cities
    if era < 4 OR (no_cities AND turn < 50):
        redirect to mercs (case 1)

    // Suppress for weaker civs
    if NOT human AND owner_ranking < best_ranking:
        if rand() % 8 < difficulty:
            redirect to tribe (case 0)

    // Skip if barbarian already on this continent
    barbarian_present = false
    if difficulty < 3:
        if turn < 30: goto spawn_barbs
        if turn < 50:
            if rand() % 2: goto spawn_barbs
    if turn > 49:
        if turn > 74: goto spawn_full
        if rand() % 2 == 0: goto spawn_barbs
    barbarian_present = true

    show "SURPRISEBARB"
    // Spawn barbarian units on adjacent tiles
    for i in 0..7:

        nx = wrap_x(x + dx[dir])
        ny = y + dy[dir]
        if valid(nx, ny) AND no_unit_at(nx, ny) AND no_city_at(nx, ny) AND NOT ocean(nx, ny):
            terrain = get_terrain(nx, ny)
            // Select barbarian unit type based on era
            if terrain.move_cost < 3:
                barb_type = late_era_type  // from tech progression
            else:
                barb_type = early_era_type

            barb_idx = spawn_unit(barb_type, BARBARIAN=0, nx, ny)
            if barb_idx >= 0:
                // Copy visibility from hut tile
                barb.visibility |= tile_visibility
                refresh(nx, ny)

            if barbarian_present: return  // only 1 barbarian if flagged

        // Step amount based on city count (skip some directions for small civs)
        step = clamp(4 - civ.city_count, 1, 4)
        i += step

case 4 (SCROLLS - Tech advance):
    if turn == 0 OR has_tech(owner, UNIVERSITY=0x26):
        redirect to gold (case 2)

    // Find a random researchable tech
    attempts = 0
    tech_roll = rand() % 100
    do:
        can_research = check_available_tech(owner, tech_roll)
        if NOT can_research:
            tech_roll = (tech_roll + 1) % 100
        else:
            show "SURPRISESCROLLS"
            grant_tech(owner, tech_roll, ...)
            tech_roll = -1  // done
        attempts++
    while attempts < 999 AND tech_roll >= 0

    if tech_roll >= 0:  // failed to find tech
        redirect to tribe (case 0)

case 5 (NOMADS - Settler unit):
    if already_has_wheel:  // tech check
        redirect to gold (case 2)

    // Check settler ratio: if settlers > cities/8, redirect to scrolls
    settler_count = count_settlers_without_home(owner)
    if settler_count > civ.city_count / 8:
        redirect to scrolls (case 4)

    show "SURPRISENOMADS"
    settler_idx = spawn_unit(SETTLER=0, owner, x, y)
    if settler_idx >= 0:
        unit[settler_idx].home_city = 0xFF  // no home city
    refresh(x, y)
