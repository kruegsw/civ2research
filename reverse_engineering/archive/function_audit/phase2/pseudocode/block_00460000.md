### FUN_00460129 — `ai_diplomacy_negotiate`

**Address**: 0x00460129 | **Size**: 16263 bytes (xlarge)
**Category**: AI.diplomacy | **Porting**: PA

**Parameters**: (civA: int, civB: int, param3: u4, param4: u4, isAiInitiated: int)

This is the **master AI diplomacy negotiation function** — the single largest function in this block. It orchestrates the complete diplomatic encounter between civA (initiating) and civB (responding). The function covers tribute demands, tech demands, alliance proposals, ceasefire/peace negotiations, senate overrides, war declarations, and patience tracking.

```
function ai_diplomacy_negotiate(civA, civB, param3, param4, isAiInitiated):
    // ── Guard & Setup ──
    diplomacy_in_progress = 1
    if multiplayer_flag != 0:
        if check_visibility(civA, civB) == 0:    // FUN_004fbe84
            diplomacy_in_progress = 0
            return

    DAT_0064b120 = civA        // store negotiating civ A
    DAT_0064b110 = civB        // store negotiating civ B
    DAT_0064b0f0 = 0           // reset negotiation state

    if isAiInitiated == 0:
        civ[civB].patience = 0                   // reset patience for human-initiated

    // ── Pre-negotiation Evaluation ──
    DAT_00626a1c = 0; DAT_00626a20 = 1
    call FUN_0045705e(civA, civB)                // evaluate diplomacy: compute tribute, attitude, tech desire
    DAT_00626a1c = 1; DAT_00626a20 = 0

    ceasefireWasActive = (treaty[civA][civB] & 0x02) != 0     // ceasefire flag
    hadNoContact = (treaty[civA][civB] & 0x01) == 0           // no contact flag

    if isAiInitiated == 0:
        // Human initiated: record turn of contact, clear 0x40002 flags
        civ[civB].lastContact[civA] = currentTurn
        clear_treaty_flags(civB, civA, 0x40002)
    else:
        // AI initiated
        if civ[civB].patience < 0:
            civ[civB].patience = 0
        if treaty[civB][civA] has bit 8 (alliance):
            civ[civB].patience = max_patience(civA, civB)    // FUN_00456f8b
        if treaty[civA][civB] bit 0x10000 set:
            civ[civB].patience = 0
            clear_treaty_flags(civB, civA, 0x200000)
        else:
            // Check nuclear first strike condition
            if difficulty > 3 AND bothCivsHaveNukes AND civA_nukes < civB_nukes AND isAtWar:
                set_treaty_flags(civB, civA, 0x200000)    // nuke talk flag

    isAiInitiatedFlag = (isAiInitiated != 0) ? 1 : 0

    // ── AI Kick-Out Checks (AI initiated only) ──
    if isAiInitiated != 0:
        if treaty[civA][civB] has bit 0x200000 (hatred):
            show_diplomat_screen(civA, civB)       // FUN_00458a3b
            set_mood(4)                            // FUN_004941ee
            show_dialog("OUTAHERE")                // refuse to talk
            goto cleanup

        // Check patience exhaustion + ambassador recall
        patience = civ[civB].patience
        maxPatience = get_max_patience(civA, civB)
        if patience >= maxPatience AND treaty[civB][civA] has bit 0x80:
            show_dialog("OUTAHERE" or "OUTAHEREALLY")
            goto cleanup

        call handle_reputation_check(civA, civB, param3, param4)  // FUN_00458df9

    // ── Set meeting flags ──
    set_treaty_flags(civA, civB, 0x800000)           // meeting in progress
    hadGiftsBefore = (treaty[civA][civB] & 0x400000) != 0
    clear_treaty_flags(civB, civA, 0x400000)

    // ── Tutorial / Scenario Override ──
    if scenario_flags & 0x80 AND special_scenario_bit:
        // Hard-coded civ pairs (3,6), (3,1) get forced diplomacy results
        DAT_0064b104 = 0; DAT_0064b124 = -1; DAT_0064b118 = 0

    // ── Gift Exchange Phase ──
    call handle_exchange_gift(civA, civB, param3, param4, 1, 0)

    if wantsNothing AND !isAiInitiated AND hasTreatyLevel >= ceasefire AND attitudeScore > 2:
        goto cleanup       // nothing to discuss

    // ── Show Diplomat Screen ──
    call show_diplomat_screen(civA, civB)

    // ── Validate third-party alliance target ──
    if 0 < thirdPartyCiv < 8:
        if civ not alive: thirdPartyCiv = -1
    if allied AND atWarWithThirdParty AND wantsToJoinWar:
        thirdPartyCiv = -1   // can't ask ally to join war against own ally

    // ═══ CANCEL TREATY PROPOSAL ═══
    if thirdPartyCiv > 0 AND (wantsToCancel OR allied):
        // Determine treaty level with third party
        treatyLevel = 2  // default: propose break contact
        if at peace with third party: treatyLevel = 1
        if allied with third party: treatyLevel = 0

        show civ name for third party
        show dialog "CANCELTREATY"

        if player accepts (result == 0):
            if not allied: adjust_attitude(civB, civA, 5)
            else:
                if wantsToCancel: adjust_attitude(civB, civA, 20)
                shouldBreakAlliance = true
            civ[civB].patience++
        else:
            // Player cancels treaty with third party
            call cancel_treaty(civA, thirdPartyCiv, civB)
            gaveGift = true
            reset demands
            // 50% chance: if ceasefire expired, renew it
            if random() & 1 == 0 AND ceasefireWasActive:
                set_treaty_flags(civB, civA, 2)   // ceasefire

    // ═══ ALLIANCE BREAK CHECK ═══
    if allied with civB:
        // Check attitude divergence: if (personalityScore - treatyViolations) > 4
        if attitudeDifference > 4 AND !justGaveGift:
            shouldBreakAlliance = true

        // Tech demand while allied
        if techDemandId >= 0:
            // Check relative tech/treasury levels
            // If weaker in tech but richer, drop demand
            // If stronger civ refuses: drop demand

        if wantsWarButAllied AND mutualDefenseActive:
            shouldBreakAlliance = true
            tributeAmount = 0

        // Scenario override: hardcoded civ pair (6,7)
        if scenario: tributeAmount = 0; techDemandId = -1; shouldBreakAlliance = true

    // ═══ TECH MAP CHECK ═══
    // Check if we have techs target lacks (for map exchange)
    wantsMap = false
    if !hasTech(civA, Alphabet[0x35]) AND !hasTech(civA, AdvancedFlight[0]):
        if hasTech(civA, Literacy[0x11]) OR hasTech(civA, MachineTools[0x51]):
            if !hasTech(civB, same): wantsMap = true
        elif hasTech(civA, Pottery[0x23]) AND !hasTech(civB, Pottery):
            wantsMap = true
    // If civB is more advanced in tech AND has Literacy: wantsMap = false

    // ═══ TRIBUTE DEMAND ═══
    if tributeAmount > 0 AND (wantsToExtort OR shouldBreakAlliance)
       AND tributeAmount <= civA.treasury:
        if allied: tributeAmount = min(civA.treasury / 2, max(100, tributeAmount))

        show civ name, ruler, government
        show "TRIBUTE" dialog

        result = show_options(0, 3)   // 0=refuse, 1=accept, 2=war, 3=...

        // Override: if weaker militarily, downgrade to acceptance
        if result == 3 AND civB_military < civA_military:
            result = 2
        if result == 2 AND !isMultiplayer: result = 1
        if wantsMap: result = 8
        if allied: result = 9

        if result == 1:    // Player pays tribute
            civA.gold -= tributeAmount
            civB.gold += tributeAmount
            wantsNothing = 0
            gaveGift = true
            call diplomacy_event(1)
            civ[civB].patience--
            adjust_attitude(civB, civA, -calculateAttitudeFromGold(tributeAmount))
            show_diplomat_screen(civA, civB)
            if ceasefireWasActive: set_treaty_flags(civB, civA, 2)

        tributeAmount = 999    // mark as handled

    // ═══ ALLIANCE ATTITUDE CHECK ═══
    if allied AND calc_attitude(DAT_0064b114) > 3:
        shouldBreakAlliance = true

    // ═══ TECH DEMAND ═══
    if (wantsToExtort OR shouldBreakAlliance) AND techDemandId >= 0
       AND tributeAmount != 999:

        show tech name
        show civ name/ruler
        show "TAKECIV" dialog
        result = show_options(0, 2)

        if wantsMap AND result < 2: result = 8
        if allied: result = 9

        if result == 1:    // Player gives tech
            grant_tech(civB, techDemandId, civA, 0, 0)
            techValue = calc_tech_value(civB, techDemandId)
            adjust_attitude(civB, civA, -(techValue * 3 / 2))

            // Check for bonus "over a barrel" demand
            if hasBonusTech AND wantsMore AND bonusTechId >= 0:
                show "OVERABARREL" dialog
                if accepted: grant bonus tech too

            gaveGift = true
            if ceasefireWasActive: set_treaty_flags(civB, civA, 2)
            civ[civB].patience--
        else if allied:
            // Refused allied tech demand: attitude penalty
            adjust_attitude(civB, civA, personalityScore + 2 - treatyViolations)
            civ[civB].patience++

    // ═══ ALLIANCE BREAK EXECUTION ═══
    if allied AND (wantsMore OR attitude > 49 OR mutualDefense) AND !gaveGift:
        call handle_reputation_check
        show_diplomat_screen
        set_mood(4)
        if tributeAmount == 999:
            show "BEGONE0"    // got tribute but wants more
        else:
            show "BEGONE1"    // no tribute, just demanding
        call break_alliance(civB, civA)
        record last contact turn
        if mutualDefense AND difficulty > 3:
            shouldDeclareWar = 1
        goto cleanup

    // ═══ PROVOCATION (WAR DECLARATION THREAT) ═══
    // Conditions: wants something, tribute > 50, has peace or (should_declare_war==false AND score > 0)
    if wantsMore AND tributeAmount > 50 AND (atPeace OR (!should_declare_war AND goodScore)):
        show "PROVOKE" dialog
        set_treaty_flags(civB, civA, 0x2000)     // provoked flag
        if attitude < 75: set_attitude(civB, civA, 75)
        record last contact turn
        goto cleanup

    // ═══ OUTRIGHT REJECTION ═══
    if wantsMore AND tributeAmount == 999:
        show "REJECT0" — outright rejection
        set provoked flag, set attitude >= 75
        goto cleanup

    if wantsMore AND (score < 0 OR nuclearThreat OR isAiInitiated):
        show rejection dialog
        set provoked flag, attitude >= 75
        goto cleanup

    if wantsMore: goto cleanup   // nothing more to say

    // ═══ ACCURSED NOTIFICATION ═══
    if wantsToComplain AND !allied:
        show "ACCURSEDUN" or "ACCURSEDWALL" (Great Wall)

    // ═══ ALLIANCE PROPOSAL ═══
    if atPeace AND !allied AND !alreadyAllyOffer:
        // Count existing alliances for civA
        allianceCount = count_alliances(civA)

        // Complex multi-condition check:
        // - Both civs difficulty < 6
        // - thirdPartyCiv not active
        // - Turn > 200
        // - At most 0 existing alliances
        // - attitude < 25 OR (attitude < 50 AND have third party to offer)
        // - Various reputation/attitude thresholds

        if all conditions met:
            show "PROPOSEALLIANCE" dialog
            if player accepts:
                // Check for "join war against third party" offer
                if thirdPartyCiv >= 0:
                    show "PERHAPSSOLIDARITY" — join war against thirdPartyCiv
                    if player accepts:
                        cancel_treaty(civA, thirdPartyCiv, civB)
                        set declared_war flag on thirdPartyCiv
                    else:
                        adjust_attitude(civB, civA, 10)
                        if conditions: show "PERHAPSDIDNTPROVE"

                if shouldFormAlliance:
                    call form_alliance(civA, civB)        // FUN_0045a535
                    allianceFormed = true

    // ═══ CEASEFIRE NEGOTIATION (when at war) ═══
    if atWar:
        show civ name
        if gaveGift:
            show "GRANTCEASE" — granting ceasefire after receiving tribute
        elif noComplaints AND !hadGiftsBefore:
            show "PROPOSECEASE" — proposing ceasefire
        else:
            show "WALLCEASE" — Great Wall ceasefire

        if player accepts (result != 0):
            // Senate/wonder override check
            if hasGreatWall(civB) OR hasUnitedNations(civB):
                call check_senate_override(civA, civB)    // FUN_0055bef9
                if senate overrides: show "OVERRULECEASE"
            else:
                call establish_ceasefire(civA, civB)

            // Calculate random demands
            demandChance = (personalityScore - treatyViolations + 1)
            if has embassy: demandChance *= 2

            if random chance passed (random % demandChance == 0):
                // Demand tech for ceasefire
                if attitudeScore > 3 AND hasTechToDemand:
                    show "GIVECIV" — demand tech for peace
                    if accepted: grant_tech; gotConcession = true

                // Demand gold for ceasefire
                if still negotiating:
                    cashDemand = clamp(attitudeScore * 2 - 4, 0, civB.gold / 50) * 50
                    if hasGreatWall/UN: cashDemand /= 2
                    if cashDemand > 0:
                        show "GIVECASH"
                        if accepted: transfer gold; gotConcession = true

                // GROVEL — demand everything (complete capitulation)
                if still negotiating AND attitudeScore > 0 AND cities < 2 AND !hasGreatWall/UN:
                    show "GROVEL" — surrender all gold + all unknown techs
                    if accepted:
                        civA.gold += civB.gold; civB.gold = 0
                        for each tech civB has that civA doesn't: grant to civA
                        gotConcession = true

                if gotConcession: record last engagement turn

            if still no deal:
                show "CONTINUEWAR"
                record last contact turn
                goto cleanup

            // Ceasefire accepted
            call establish_ceasefire(civA, civB)  // FUN_0045a7a8
            show_diplomat_screen
            if attitude very hostile (diff > 6):
                show "WORTHLESS" — dismissive remark

    // ═══ PEACE TREATY NEGOTIATION ═══
    if !atPeace:
        // Military power comparison for tone
        toneLevel = 3   // base
        if civA_military * 2 < civB_military: toneLevel = 2
        if civA_military * 1.5 < civB_military: toneLevel--
        if civB_military < civA_military: toneLevel++
        if civB_military * 2 < civA_military: toneLevel++

        if conditions for peace proposal met:
            show "PROPOSEPEACE"
            if player accepts:
                // Senate/wonder override
                if hasGreatWall/UN: check_senate_override

                if override:
                    show "WALLOVERPEACE" or "UNOVERPEACE"
                else:
                    if !(treaty has 0x400000 bit):
                        call establish_peace(civA, civB)  // FUN_0045fd67

                // Gold demand for peace
                cashDemand = clamp(attitudeScore*2-4, 0, civB.gold/50) * 50
                if cashDemand > 0:
                    show "CASHFORPEACE"
                    if accepted: transfer gold; gotConcession = true

                if deal made:
                    call finalize_peace(civA, civB)    // FUN_0045a6ab
                else:
                    show "NOPEACE"

    // ═══ GREETING / ATTITUDE DISPLAY / INTERACTIVE PHASE ═══
    if shouldShowGreeting:
        if isAiInitiated:
            if noContact:
                show_diplomat_screen

        if atPeace AND !isAiInitiated:
            // War declaration check
            if attitude very hostile (diff > 6):
                if hasEmbassy AND (hasAlliances OR wantsWar OR civA weaker):
                    shouldDeclareWar = 1

        if hadNoContact AND isAiInitiated:
            call handle_reputation_check

        // Senate forced peace/ceasefire
        if !shouldDeclareWar:
            check for GreatWall/UN senate override
            if overrides AND !atPeace:
                show "SENATEPEACE" — force peace
                set_treaty_flags(civB, civA, 4)   // peace
                record contact + 16 turns grace
            if overrides AND atWar:
                show "SENATECEASE" — force ceasefire
                set_treaty_flags(civB, civA, 2)
        else:
            // AI decides to declare war
            show_diplomat_screen

            // ── GREETING DIALOGS ──
            if tributeHandled OR gaveGift:
                // Show welcome/howdy with treaty-level variants
                if !isAiInitiated:
                    pick label from 0x6f/0x70/0x71 based on military comparison
                    show "WELCOME" / "WELCOMEPEACE" / "WELCOMEALLY"
                else:
                    if patience < 2:
                        show "HOWDY" / "HOWDYPEACE" / "HOWDYALLY"
                    else:
                        show "DOODY" / "DOODYALLY"  (impatient greeting)
            else:
                // Attitude display
                show "ATTITUDE" / "ATTITUDEPEACE" / "ATTITUDEALLY"

            // ── CITY PLEA (contested cities) ──
            if !atPeace AND !allied:
                call check_contested_cities(civB, civA)
                if contested > 0:
                    show "PLEASECITY" or "PLEASECITIES"

            // ── ALLY PLEA/BRAG (alliance mutual defense) ──
            if civB allied with civA:
                // Check continent strength vs mutual enemy
                if mutual enemy outperforms us: show "ALLYPLEA"
                elif we outperform enemy: show "ALLYBRAG"

            // ── INTERACTIVE NEGOTIATION LOOP ──
            do:
                if civ[civB].patience < 0: reset to 0

                result = show_negotiation_menu(civA, civB)   // FUN_0045fe19

                if result == 0: kicked_out = true; goto cleanup
                if result == 7: show_trade_dialog(civA, civB)   // FUN_0045dd7f
                elif result == 8: show_gift_menu(civA, civB)
                else:
                    result2 = handle_negotiation_choice(civA, civB, result)  // FUN_0045b4da
                    if result2 != 0: goto cleanup

                civ[civB].patience++
            while civ[civB].patience < max_patience(civA, civB)

            // Patience exhausted
            show "PATIENCE" / "PATIENCEALLY"

    // ═══ CLEANUP ═══
    cleanup:
    clear_treaty_flags(civB, civA, 0x10000)      // clear meeting flag

    if shouldDeclareWar == 0 OR (treaty[civB][civA] has 0x2008):
        treaty[civB][civA] &= ~0x20              // clear hatred
    else:
        treaty[civB][civA] |= 0x20               // set hatred (war declared)

    if !kicked_out:
        civ[civB].patience++

    if gotConcession OR hadGiftsBefore:
        // Boost patience to max + concession count
        while patience < max_patience + bonus:
            civ[civB].patience++

    call clear_diplomat_screen()                 // FUN_0045918e

    // ── CEASEFIRE EXPIRY NOTIFICATION ──
    if ceasefireWasActive AND !(treaty has ceasefire) AND !(treaty has war/provoked):
        show "CEASEEXPIRE"

    // ── AMBASSADOR OFFER ──
    if difficulty == 0 AND !atWar AND !hasAmbassador AND !hasIntelligence
       AND civB_has_tech(Writing):
        show "AMBASSADORS"
        treaty[civB][civA] |= 0x80               // ambassador flag
        call grant_embassy(civA, civB)           // FUN_0043060b

    // ── TUTORIAL: SCHISM ──
    if tutorial_enabled AND isAiInitiated AND atWar
       AND civB_difficulty > civA_difficulty + 1 AND cities > 4:
        show "SCHISM"

    diplomacy_in_progress = 0
```

### FID_conflict___E31 @ 0x00467620 — `crt_init_static_1`

```
        ch = toupper(*str++)
        if isdigit(ch):
```

```
function load_bitmap(surface, filename, palStart, palCount, loadPalette) -> int:
    file = fopen(filename, "rb")
    if file == NULL: return 0

    // Read BMP file header (14 bytes)
    read(fileHeader, 14)
    if fileHeader.magic != 0x4D42: error("Invalid bitmap file")

    // Read info header
    read(infoHeader)

    // Read palette (1024 bytes = 256 * 4 RGBQUAD)
    read(palette, 0x400)

    // Validate: must be 640x480, 8-bit
    if width != 640 OR height != 480 OR bitsPerPixel != 8:
        error("Image not 640 x 480 x 256 color")

    // Apply palette if requested
    if loadPalette:
        for i = palStart to palStart + palCount:
            set_palette_entry(i, palette[i-palStart])
        animate_palette(palStart, palCount)

    // Setup surface dimensions
    set_surface_size(width, height)

    // Read pixel data
    if compression == 0:     // uncompressed
        for y = height-1 downto 0:
            ptr = get_scanline(0, y)
            for x = 0 to width:
                byte = read_byte()
                *ptr++ = byte + palStart    // offset to game palette

    elif compression == 1:   // RLE8
        y = height - 1
        ptr = get_scanline(0, y)
        while true:
            count = read_byte()
            if count == 0:
                escape = read_byte()
                if escape == 0:     // end of line
                    y--; ptr = get_scanline(0, y)
                elif escape == 1:   // end of bitmap
                    break
                elif escape == 2:   // delta
                    dx = read_byte(); dy = read_byte()
                    y -= dy; x += dx
                    ptr = get_scanline(x, y)
                else:               // absolute mode
                    for i = 0 to escape:
                        *ptr++ = read_byte() + palStart
                    if escape & 1: read_byte()   // padding
            else:                   // run
                value = read_byte()
                for i = 0 to count:
                    *ptr++ = value + palStart

    elif compression == 2:
        error("Incompatable compression mode")

    fclose(file)
    return success
```

## Cross-Reference: Treaty Flag Bits

From analysis of `clear_treaty_flags` and `set_treaty_flags`:

| Bit | Hex | Meaning |
|-----|-----|---------|
| 0 | 0x01 | Contact established |
| 1 | 0x02 | Ceasefire |
| 2 | 0x04 | Peace treaty |
| 3 | 0x08 | Alliance |
| 4 | 0x10 | Embassy |
| 5 | 0x20 | Hatred / at war (in byte+1, i.e., bit 13) |
| 7 | 0x80 | Ambassador recall |
| 11 | 0x0800 | Unknown (cleared by provoke) |
| 12 | 0x1000 | Unknown (cleared by provoke) |
| 13 | 0x2000 | Provoked |
| 17 | 0x010000 | Meeting in progress |
| 18 | 0x040000 | Recent gift flag |
| 19 | 0x080000 | Unknown |
| 21 | 0x200000 | Nuke talk / hatred modifier |
| 23 | 0x800000 | Active negotiation |

**Cascade rules**:
- Setting alliance (0x08) also sets peace (0x04)
- Setting any treaty (0x0E) clears war flags (0x2A60)
- Setting provoked (0x2000) clears all treaties (0x0E) and adds hatred (0x200000)
- Clearing peace (0x04) also clears alliance (0x08)
- Clearing provoked (0x2000) also clears 0x1800
- Clearing contact (0x01) also clears provoked (0x2000)
