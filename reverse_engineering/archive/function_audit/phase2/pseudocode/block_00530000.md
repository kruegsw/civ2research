## Function 13: FUN_005312E4 — `ai_find_best_settle_dir`

- **Address**: 0x005312E4 | **Size**: 643 bytes
- **Category**: AI.settle
- **Porting status**: PA (ported in cityai.js as part of settler logic)
- **Calls**: `wrap()`, `is_valid_tile()`, `is_ocean()`, `get_city_at()`, `get_continent()`, `is_at_war()`
- **Called by**: `ai_process_civ_turn`, `ai_unit_turn_master`
- **Key globals**: `DAT_00628350/60` (8-direction dx/dy arrays), `DAT_00655b0b` (humanPlayerBitmask)

### Pseudocode

```
function ai_find_best_settle_dir(unitIdx):
    unitX = units[unitIdx].x
    unitY = units[unitIdx].y
    owner = units[unitIdx].owner
    bestScore = -1
    bestDir = -1

    // Evaluate 9 candidate tiles (8 neighbors + center at index 8)
    for dir = 0 to 8:
        candX = wrap(unitX + dx8[dir])
        candY = unitY + dy8[dir]

        if not is_valid_tile(candX, candY): continue
        if not is_land(candX, candY): continue          // must be land
        if dir != 8:                                     // neighbors only:
            if get_city_at(candX, candY) >= 0: continue  // can't build next to city

        // Score = count of workable land tiles around candidate
        score = 0
        for adj = 0 to 7:
            adjX = wrap(dx8[adj] + candX)
            adjY = dy8[adj] + candY
            if not is_valid_tile(adjX, adjY): continue
            if is_land(adjX, adjY): continue    // skip — already counted as candidate
            if get_city_at(adjX, adjY) >= 0: continue

            // Check for enemy cities: no enemy city unless war + center tile
            tileOwner = get_tile_owner(adjX, adjY)
            if tileOwner >= 8 OR (humanBitmask & (1 << tileOwner)) == 0
               OR (treaties[tileOwner][owner] & 6) == 0:
                continent = get_continent(adjX, adjY)
                if continent >= 0 AND continent != owner:
                    cityCont = get_city_at(adjX, adjY)
                    if cityCont >= 0 AND is_at_war(owner, tileOwner) AND dir == 8:
                        score += 12      // bonus: enemy city tile reachable from center
                score += 1

        if score >= bestScore:
            bestScore = score
            bestDir = dir

    return bestDir    // -1 = no valid site found
```

### Notes — Missing from JS

Ported as part of `generateSettlerActions()` in cityai.js. The +12 bonus for enemy cities near center is present.

### Pseudocode

```
function ai_calc_continent_city_weight(civIdx, continentId):
    // Weighted city count: first 5 cities worth 3 each, next 5 worth 2, rest worth 1
    total = 0
    remaining = civData[civIdx].continentCityCount[continentId]
    weight = 3

    do:
        batch = remaining
        if batch > 5: batch = 5       // cap at 5 per weight tier
        remaining -= batch
        total += weight * batch
        weight -= 1
    while remaining > 0 AND weight > 0

    return total
```

## Function 19: FUN_00536C4C — `ai_find_nuke_target`

- **Address**: 0x00536C4C | **Size**: 1760 bytes
- **Category**: AI.nuke
- **Porting status**: PA (referenced in unitai.js nuclear role handling)
- **Calls**: `distance()`, `distance_manhattan()`, `get_unit_range()`, `get_city_at()`, `is_wonder()`, `ai_set_goto_order()`, `wrap()`, `set_treaty_flag()`
- **Called by**: ai_unit_turn_master (role 0x48)

### Pseudocode

```
function ai_find_nuke_target(unitIdx):
    owner = units[unitIdx].owner
    bestScore = -1
    bestCity = -1

    // Scan all enemy cities for best nuclear strike target
    for c = 0 to maxCities:
        if cities[c].dead: continue
        targetOwner = cities[c].owner
        if targetOwner == owner: continue

        // Skip if no SDI defense bypass possible
        if has_wonder(c, SDI_DEFENSE): continue

        // Must be large enough to be worth nuking
        if cities[c].size < 5: continue

        // Our total military strength must be >= 2/3 of theirs
        // OR treaty has nuked flag OR hatred flag
        ourStrength = civData[owner].totalAtkDefSum
        theirStrength = civData[targetOwner].totalAtkDefSum
        if NOT (theirStrength*2 <= ourStrength*3
                OR (treaties[targetOwner][owner] & NUKED_FLAG) != 0
                OR (unitTypeEverBuilt[targetOwner] == 0
                    AND theirStrength*2 <= ourStrength)):
            continue

        // Treaty check: must have contacted (0x100) but NOT allied (0x04)
        if (treaties[targetOwner][owner] & 0x104) != 0x100: continue

        // Score the target
        score = 0
        for dir = 0 to 8:   // 9 tiles including center
            tileX = wrap(cities[c].x + dx9[dir])
            tileY = cities[c].y + dy9[dir]
            if not is_valid_tile(tileX, tileY): continue
            tileOwner = get_tile_owner(tileX, tileY)
            if tileOwner > 0:
                if tileOwner == owner:
                    score -= 2     // penalty: own territory
                elif (treaties[tileOwner][owner] & 0x104) == 0x100:
                    // Enemy units present = higher value
                    for stackUnit = get_first_unit(tileX, tileY);
                        stackUnit >= 0; stackUnit = next_in_stack(stackUnit):
                        score += 1
                else:
                    score -= 99    // massive penalty: allied territory

        score += cities[c].size / 2

        if score > bestScore:
            // Verify reachability: one of our cities must be close enough
            reachable = false
            for oc = 0 to maxCities:
                if cities[oc].dead: continue
                if cities[oc].owner != owner: continue
                dist = distance_manhattan(cities[oc].x, cities[oc].y,
                                          cities[c].x, cities[c].y)
                range = get_unit_range(unitIdx)
                if dist <= range / moveSpeed:
                    reachable = true; break

            if reachable:
                bestCity = c
                bestScore = score

    // Execute strike
    if bestScore >= (10 - numOwnCities) AND unitTypeCount > 1:
        // Find adjacent empty tile to position nuke
        for dir = 0 to 7:
            adjX = wrap(cities[bestCity].x + dx8[dir])
            adjY = cities[bestCity].y + dy8[dir]
            if is_valid_tile(adjX, adjY) AND get_city_at(adjX, adjY) < 0
               AND get_tile_owner(adjX, adjY) < 0:
                teleport_unit(unitIdx, adjX, adjY, 1)
                units[unitIdx].visibility = 0

                // Set treaty flags: nuked
                if (humanBitmask & (1 << cities[bestCity].owner)) != 0:
                    if (treaties[targetOwner][owner] & ALLIANCE) == 0:
                        treaties[targetOwner][owner] |= 0x20000  // nuked
                        treaties[targetOwner][owner] &= ~0x10    // clear
                        set_treaty_event(owner, targetOwner, 0x10000)
                    else:
                        attitude[targetOwner][owner] -= 2

                return dir ^ 4    // return opposite direction

    return 8    // no target found
```

```
                        continue
```

```
                dist = distance_step(unitX, unitY, cities[c].x, cities[c].y)

                if (turnNumber & 7 + 4) <= difficulty:
                    kill_unit(unitIdx, 1)
                    return 1

        bestScore = -999
        for dir = 0 to 7:

            if not is_valid_tile(adjX, adjY): continue

            terrain = get_terrain_type(adjX, adjY)
            // Domain check: land unit on land, sea unit on sea
            if (domain == SEA) != (terrain == OCEAN): continue

            score = 0
            if role < 5:    // combat unit
                cityOwner = get_city_at(adjX, adjY)
                if cityOwner < 0 OR cityOwner == owner:

                    // Different scoring for ocean tiles
                    if has_stack(adjX, adjY):

                        else: score = -99

                    buildCity = find_nearest_city(adjX, adjY)
                    score += terrain_food[terrain] + random(0,3) + difficulty*4
                    goto COMPARE_SCORES

                    if tileOwner != owner:
                        score += 4    // prefer moving into enemy territory

                    if tile_has_improvements(adjX, adjY):
                        score += 6    // roads
                    if tile_has_fortress(adjX, adjY):
                        score += 8    // fortress

                    score += random(0, 5)

                    // Penalize: own units, hostile stack
                    if stack_on_tile and owner match:
                        score -= 20
                    elif no_enemies_on_land(unitX, unitY):
                        if stack_owner == owner: score += 99

                    // Direction bias
                    if lastDir == dir: score += 6
                    if targetDX != -3:
                        if sign(dx8[dir]) == targetDX: score += 2
                    if targetDY != -3:
                        if sign(dy8[dir]) == targetDY: score += 2

                    COMPARE_SCORES:
                    if score >= bestScore:
                        bestScore = score
                        bestDir = dir

    // Check: if on city tile and not too strong, pillage
    tileOwner = get_city_at(unitX, unitY)
    if (tileOwner < 0 OR hp > 1) OR bestScore > 98:
        // Can we sit here? Various conditions for fortifying vs moving
        if bestScore < 99 AND hp <= moveSpeed AND nearestCity >= 0:
            // Check tile for strategic value (improvements, fortress, resources)
            if tile_has_fortress OR tile_has_improvements:
                disband_unit_as_spy(unitIdx, 0)
                unit_sleep(unitIdx)
                goto FINALIZE
    else:
        bestDir = 8   // stay put

    FINALIZE:
    if bestDir == 8:
        // No movement: stay and potentially fortify
        units[unitIdx].lastDir = 0xFF
        unit_sleep(unitIdx)
        if role == SETTLER OR tile_has_fortress(unitX, unitY):
            units[unitIdx].orders = FORTIFYING
            if attribs & 0x100:
                units[unitIdx].orders = FORTIFIED
    else:
        // Move in chosen direction
        units[unitIdx].orders = 0x1B   // AI_MOVE_ADJACENT
        units[unitIdx].gotoX = wrap(unitX + dx8[bestDir])
        units[unitIdx].gotoY = unitY + dy8[bestDir]

    // Final check: if GOTO target == current position, cancel
    if units[unitIdx].orders == GOTO
       AND units[unitIdx].gotoX == unitX AND units[unitIdx].gotoY == unitY:
        units[unitIdx].orders = 0xFF

    return 0
```

```
           AND unitTypes.cost > 0:

            tileFeatures = get_tile_features(unitX, unitY)
            if (tileFeatures & 0x4C) != 0 OR (typeId == 9 AND road):
                // Fortify and wait: strategic position
                disband_as_spy(unitIdx, 0)
                unit_sleep(unitIdx)
                goto FINALIZE

    // === Explorer role: find nearby diplomats to capture ===
    if typeId == 9 AND units[unitIdx].cargo == 0:
        // Scan for enemy diplomats/settlers within 4 tiles
        for u in all_units:
            if units[u].owner != owner AND role > 4
               AND distance < 4 AND same_continent
               AND (visible to us OR same owner):
                ai_set_goto_order(unitIdx, 0x50, units[u].x, units[u].y)
                needTransport = 1
                break

    // === Explorer: pillage enemy territory ===
    if typeId == 9 AND nearestEnemyCity >= 0
       AND (humanBitmask & (1 << enemyCity.owner)) != 0
       AND not at_peace:
        dist = distance_step(unitX, unitY, enemyCity.x, enemyCity.y)
        bestPillage = -1
        for dir = 0 to 7:
            adjX, adjY = neighbor(unitX, unitY, dir)
            if not city AND tile_owner < 0:
                dist2 = distance_step(adjX, adjY, enemyCity.x, enemyCity.y)
                if dist2 <= dist AND has_improvements(adjX, adjY):
                    bestPillage = dir
        if bestPillage >= 0:
            units[unitIdx].aiRole = 0x70
            chosenDir = bestPillage
            goto APPLY_DIRECTION

    // === SETTLER TERRAFORM: Check nearby tiles for terraform opportunities ===
    if (isSettlerRole OR role == 1) AND has_tech(owner, BRIDGE_BUILDING):
        distToCity = ...
        if distToOwnCity != 0 AND nearestOwnCity == nearestEnemyCity
           AND not onFortress
           AND on_land AND not in_city:
            // Evaluate all 8 neighbors for terraform needs
            // ... complex decision tree for irrigation, road, fortress, mine ...
            // Based on: terrain type, food yield, city proximity, threat level
            // Goal: prioritize irrigation for food-poor tiles, roads for connectivity,
            //        fortresses for strategic chokepoints

            oceanCount = 0; hasNorth = 0; hasSouth = 0; hasWest = 0; hasEast = 0
            for dir = 0 to 7:
                check ocean adjacency in 4 cardinal directions
                check extended ocean (2 tiles away) in 4 directions

            // Narrow passage detection
            if (hasNorth + hasSouth > 1) AND (hasWest == 0 OR hasEast == 0):
                shouldFortress = 2   // narrow NS passage
            if (hasWest + hasEast > 1) AND (hasNorth == 0 OR hasSouth == 0):
                shouldFortress = 2   // narrow EW passage

            if shouldFortress:
                if isSettlerRole:
                    create_production_goal(owner, unitX, unitY, 1, 2)
                else:
                    create_production_goal(owner, unitX, unitY, 0x15,
                                            shouldFortress + (distToCity < 4))

    // === SETTLER: goto-arrived check ===
    if role == 5 AND orders == GOTO:
        if get_city_at(gotoX, gotoY) >= 0:
            continueSailing = 1   // settler going to city = keep going

    // If settler on land, check terraform queue
    if isSettlerRole AND not autoAttack:
        if not gotoBroken AND on_land AND distToOwnCity > 0:
            ...settler terraform and city-founding decision tree...

    // === GOTO CONTINUATION CHECK ===
    if orders == GOTO:
        if role == 2 AND in_zoc: goto FINALIZE   // defender in ZOC stays
        dist = distance(unitX, unitY, gotoX, gotoY)
        if dist > 1:
            if role < 5 AND in_zoc: needTransport = 1; break = false
            goto CHECK_GOTO_DEST

        CHECK_GOTO_DEST:
        destOwner = get_tile_owner(gotoX, gotoY)
        if destOwner >= 0 AND destOwner != owner:
            if role != 4 OR (hp > 1 OR needTransport):
                goto CANCEL_GOTO

        CANCEL_GOTO:
        if at_war(destOwner, owner):
            break_goto = false
            units[unitIdx].orders = 0xFF
            needTransport = 1

    // === 8-DIRECTION MOVEMENT EVALUATION ===
    // This is the final movement decision for units not handled above
    bestScore = -999
    bestAbsScore = -999
    chosenDir = 8
    exploration = 0
    combatTarget = 0

    // Determine if unit can explore (wants to see new tiles)
    if in_zoc(unitX, unitY, owner) == 0:
        if needTransport == 0: exploration = 1
        else: exploration = 0
    if (gameFlags & 0x80) != 0 AND (DAT_0064bc60 & 8): exploration = 0

    // Adjust role for defense→attack if outnumbered
    if role == 1:
        if unitTypes[typeId].defense < unitTypes[typeId].attack
           AND (continentFlags & STRONG_THREAT) == 0:
            role = 0   // switch to attack
        if unitTypes[typeId].defense <= unitTypes[typeId].attack
           AND continentThreat == 0:
            role = 0

    // === EVALUATE EACH DIRECTION ===
    for dir = 0 to 7:
        adjX, adjY = neighbor(unitX, unitY, dir)
        if not is_valid_tile(adjX, adjY): continue

        tileOwner = get_tile_owner(adjX, adjY)
        terrain = get_terrain_type(adjX, adjY)
        isOcean = (terrain == OCEAN)

        // Domain check
        if (isOcean == false) OR (domain != 0): skip conditions

        stackUnit = get_first_unit(adjX, adjY)
        // Skip air units when evaluating ground threats
        while stackUnit >= 0 AND is_air(stackUnit) AND has_fuel(stackUnit):
            stackUnit = get_next_in_stack(stackUnit)

        // PASSABILITY: complex conditions for entering tile
        // - hasEnemyOnTile: can't enter if enemy + no attack capability
        // - domain mismatch: land on ocean or vice versa
        // - stacking limits: can't stack too many units
        // - treaty restrictions: can't enter allied/peaceful territory
        if not passable: continue

        score = 0

        // === SETTLER SCORING ===
        if isSettlerRole:
            if terrain == OCEAN AND in_zoc: skip
            score += random(0, 4)

        // === EXPLORER SCORING (no target set) ===
        elif units[unitIdx].visibility == 0 AND terrain != OCEAN:
            if role == 0:
                score += random(0, 2) - terrain_move_cost * 2
            else:
                score += random(0, 2) - terrain_defense

        // === COMBAT SCORING (has target or enemy present) ===
        else:
            score += random(0, 4)

            if stackUnit < 0 OR tileOwner == owner:
                // Friendly or empty tile
                if role < 6:   // combat role
                    score += terrain_defense * 4
                else:
                    // Non-combat: penalize based on terrain cost
                    score += 6 - (terrain_cost - 1) * moveSpeed
                    // Proximity to enemy city bonus/penalty
                    if distToEnemyCity < 4 AND nearestEnemyCity != nearestOwnCity:
                        dist = distance(adjX, adjY, enemyCity.x, enemyCity.y)
                        if dist == 1: score += terrain_cost > moveSpeed ? -4 : 8
                        elif dist == 2 AND road_present: score += 4

            else:
                // ENEMY UNIT ON TILE
                tileOwner = units[stackUnit].owner
                if tileOwner != owner:
                    isCombat = true
                    captureable = false

                    // Check if we can attack
                    if isSettlerRole OR role == 7 OR
                       (domain == SEA AND target is air AND our atk < 3) OR
                       (on_ocean AND domain == 0 AND not amphibious):
                        skip   // can't attack

                    if (treaties[tileOwner][owner] & 6) == 0:
                        // No peace: evaluate attack
                        if is_human(tileOwner): skip some checks

                        // COMBAT EVALUATION: compare our attack vs their defense
                        combatResult = evaluate_combat(unitIdx, dir, 0)
                        stackHP = count_enemy_stack_hp(stackUnit, 0)

                        adjustedResult = (stackHP + 1) * combatResult
                                         / unitTypes[typeId].firepower

                        // Score based on relative strength
                        if in_city(adjX, adjY): adjustedResult *= 3
                        if role == 2 AND target is air: adjustedResult *= 2

                        if needTransport == 0:
                            // Scale by role preference
                            if role == 0:
                                if continentThreat == 0: adjustedResult *= 2
                                if atk > def * 2: adjustedResult *= 2; captureable = true
                                if has_attack_goal_nearby: adjustedResult *= 2
                                if enemy_wounded: adjustedResult *= 2; captureable = true
                                if flanking_unit: adjustedResult *= 3
                            else:
                                adjustedResult /= 2
                                if target.atk <= our.def: adjustedResult = 0

                            // Explorer special: capture weak units
                            if typeId == 9 AND target is settler:
                                captureable = 2

                            // Barbarian bonus
                            if tileOwner == 0 AND role == 7:
                                if not at_peace(barbarian): adjustedResult /= 2
                            elif tileOwner == 0 AND role > 4:
                                adjustedResult += 12
                            elif role == 6: adjustedResult *= 2

                            // Final threshold check
                            if adjustedResult >= (tileOwner == 0 ? 12 : 6):
                                score += adjustedResult * 4
                                goto SCORE_DONE

                            // Too weak to attack: consider picking up/disbanding
                            score = -999
                            if conditions for picking up weak friendly:
                                pick_up_unit(stackUnit, owner)

                        goto SKIP_TILE

                    // AT PEACE: check for treaty violations
                    if at_peace AND role == 6:
                        // Diplomat can enter for espionage
                        score = 60
                        ...
                    else:
                        goto SKIP_TILE

                else:
                    // OWN UNIT ON TILE: adjust score
                    score -= unitTypes[stackUnit.typeId].defense

        SCORE_DONE:
        // Additional adjustments
        if distToOwnCity != 0 AND domain == SEA:
            // Naval: prefer heading toward continent with goals
            contDir = get_continent_direction(adjX, adjY)
            score += continent_matching_bonus

        // Penalty for moving away from assigned GOTO target
        if domain == 1 AND target is ocean: score += penalty

        // Exploration bonus: unseen tiles are valuable
        if exploration:
            // Check 4-tile line in this direction for visibility
            farX = unitX + dx8[dir] * 4
            farY = unitY + dy8[dir] * 4
            if is_valid(farX, farY) AND not visible(farX, farY, owner):
                if not is_ocean(farX, farY): score += 8
                else: score += 2
            // Check neighbors of target for visibility
            for n = 0 to 7:
                nnX, nnY = neighbor(farX, farY, n)
                if valid AND not visible AND (not ocean OR domain == SEA):
                    score += 2
                if has_enemy_civ(nnX, nnY): score -= 2
                if isSettlerRole AND terrain_food > 0:
                    score += terrain_food_bonus

        // Edge penalty: don't go to map edge
        if adjY == 0 OR adjY == mapHeight - 1: score /= 3

        // Damage-weighted scoring: if weak, reduce attack desire
        if isCombat AND get_hp(unitIdx) < moveSpeed:
            if get_city_at(adjX, adjY) < 0:
                score = score * (get_hp(unitIdx, 0) / get_hp(unitIdx, 3))

        if score > bestScore:
            bestScore = score
            chosenDir = dir
            combatTarget = isCombat
            combatStyle = captureable
        if absoluteScore > bestAbsScore:
            bestAbsScore = absoluteScore

    // === POST-DIRECTION CHECKS ===
    if combatTarget AND get_hp(unitIdx) < moveSpeed - combatStyle:
        chosenDir = 8   // too weak to attack, cancel
    if bestAbsScore != bestScore:
        chosenDir = 8   // absolute score disagrees, cancel
    if chosenDir == 8:
        combatTarget = 0

    if needTransport AND combatTarget == 0:
        goto FINALIZE   // already set up transport GOTO
    if wantRetreat AND combatTarget == 0:
        chosenDir = 8

    units[unitIdx].aiRole = 0x39  // standard move role

    // === APPLY DIRECTION ===
    APPLY_DIRECTION:
    if chosenDir == 8:
        // No movement: idle
        units[unitIdx].lastDir = 0xFF
        units[unitIdx].orders = 0xFF
    else:
        // Set AI_MOVE_ADJACENT
        adjX = wrap(unitX + dx8[chosenDir])
        adjY = unitY + dy8[chosenDir]
        if not is_valid_tile(adjX, adjY):
            units[unitIdx].orders = 0xFF
            unit_sleep(unitIdx)
        else:
            units[unitIdx].orders = 0x1B   // AI_MOVE_ADJACENT
            units[unitIdx].gotoX = adjX
            units[unitIdx].gotoY = adjY

    // === FINALIZE ===
    FINALIZE:
    // Final order cleanup
    if orders in [0xFF, 0x10, 0x01, 0x02]:
        if orders == 0x10: aiRole = 0x30   // AI_WAIT
        if domain == NAVAL:
            units[unitIdx].orders = 0xFF
            units[unitIdx].cargo += moveSpeed
            return 1
        unit_sleep(unitIdx)

        if domain == 0 AND role != SETTLE:
            units[unitIdx].orders = FORTIFYING
            if attribs & 0x100: units[unitIdx].orders = FORTIFIED
            if shouldExploreInstead: units[unitIdx].orders = 0xFF
        else:
            units[unitIdx].orders = 0xFF

    // Self-GOTO detection: cancel if destination == current position
    if orders == GOTO AND gotoX == unitX AND gotoY == unitY:
        units[unitIdx].attribs |= 0x80
        units[unitIdx].orders = 0xFF
        if units[unitIdx].cargo != 0:
            if role == SEA AND has_cargo(unitIdx, 4):
                unit_sleep(unitIdx)
            else:
                unit_sleep(unitIdx)

    return 0
```
