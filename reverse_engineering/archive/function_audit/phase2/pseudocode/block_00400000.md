### Signature
`void minimap_calc_viewport(void)`

### Pseudocode
```
// Calculate tile pixel sizes to fit minimap in available space
tileH = panel_height / map_height  // min 1
tileW_half = panel_width / map_width  // min 1
tile_size = min(tileH, tileW_half)
DAT_0063c804 = tile_size       // tile height in pixels
DAT_0063c800 = tile_size * 2   // tile width in pixels

// Visible columns/rows (clamped to map bounds)
vis_cols = min(map_width/2, (panel_width - tileW/2) / tileW)
vis_rows = min(map_height, panel_height / tile_size)

// Margins (centered in panel)
margin_x = clamp(panel_width - vis_cols * tileW, 0, 999) / 2
margin_y = clamp(panel_height - vis_rows * tileH, 0, 999) / 2

// Scroll offset (center on current view position)
if vis_rows < map_height:
  scroll_y = clamp(cursor_y - vis_rows/2, 0, max_scroll_y)
else:
  scroll_y = 0

scroll_x = wrap(cursor_x - vis_cols)
if non-wrapping map:
  scroll_x = clamp(scroll_x, 0, max_scroll_x)

// Parity fix: ensure scroll_x parity matches scroll_y parity
if scroll_x is odd: scroll_x -= 1
if scroll_y is odd: scroll_x += 1
```

## FUN_0040897f / mapgen_calc_fertility
**Category**: GL.mapgen | **Size**: 948b | **Ported**: P

### Signature
`void mapgen_calc_fertility(void)`

### Pseudocode
```
// Zero continent tile counters
for c = 0 to 63:
  continent_tile_count[c] = 0

// Pre-compute terrain yield scores from RULES.TXT terrain table
// For each terrain type (0..21), score = food*3 + trade + shields*2
// Adjustments for special resources (DAT_00627ccf = resource1_food, DAT_00627cce = resource2_food)
for t = 0 to 21:
  base_terrain = t % 11
  score[t] = terrain_food[t] * 3 + terrain_trade[t]
  if base_terrain != 2:  // not grassland
    score[t] += terrain_shields[t] * 2
  // Add special resource bonus
  if terrain_resource1_food[base_terrain] == -2:  // flag for +shields
    score[t] += (base_terrain == 4) ? 3 : 1  // hills get +3
  elif terrain_resource2_food[base_terrain] == -2:
    score[t] += 2

// For each tile on the map
for y = 0 to map_height - 1:
  for x = (y & 1) to map_width - 1 step 2:
    terrain = get_terrain_type(x, y)
    if terrain == PLAINS or terrain == GRASSLAND:
      total = 0
      // Scan 21-tile city radius (indices 0..20)
      for i = 0 to 20:
        nx = wrap(x + city_spiral_dx[i])
        ny = y + city_spiral_dy[i]
        if is_tile_valid(nx, ny):
          t = get_terrain_type(nx, ny)
          if t == GRASSLAND and grassland_has_shield(nx, ny):
            tile_score += 2
          // Check for special resources at neighbor
          has_resource = check_resource(nx, ny)
          if has_resource and i != 20:
            t += 11  // use resource variant score
          tile_score += score[t]
          // Check for river
          if tile_byte0[nx,ny] & 0x80:  // river bit
            tile_score += 1
          // Weighting: inner ring (i < 8) gets 2x, city center (i == 20) gets 4x
          if i < 8:
            tile_score *= 2
          elif i == 20:
            tile_score *= 4
            if river: tile_score += tile_score / 2  // river bonus on center
          total += tile_score

      // Adjust for non-shield grassland
      if terrain != PLAINS and !grassland_has_shield(x, y):
        total -= 16

      // Normalize to 1-15 range
      fertility = clamp((total - 0x78) >> 3, 1, 15)
      // Store as signed byte: value + 8, then - 16
      stored = (fertility >> 1) + 8

      // Increment continent tile counter
      continent = get_continent_id(x, y)
      if continent >= 0:
        continent_tile_count[continent] += 1

      tile_byte5[x,y] = stored - 16  // stored as signed byte

    else:
      // Non-fertile terrain: set to 0xF0 (-16 signed)
      tile_byte5[x,y] = 0xF0
```

- Calls: FUN_005b89bb (get_terrain_type), FUN_005b8931 (get_tile_ptr), FUN_005b8ee1 (check_resource), FUN_005b8a81 (get_continent_id), FUN_005ae052 (wrap_x), FUN_004087c0 (is_tile_valid), FUN_0040bcb0 (grassland_has_shield), FUN_005adfa0 (clamp)

## FUN_0040ac5a / generate_rivers
**Category**: GL.mapgen | **Size**: 1242b | **Ported**: P

### Signature
`void generate_rivers(void)`

### Pseudocode
```

do:
  // Backup terrain layer (for possible revert)
  copy_byte_layer(scratch, terrain_layer)
  attempts += 1
  river_length = 0

  // Pick random starting tile (not mountain/ocean)
  do:
    y = rand() % map_height
    x = (rand() % (map_width/2)) * 2
    if y is odd: x += 1
    terrain = *get_tile_ptr(x, y)
  while terrain == MOUNTAINS or terrain == OCEAN

  // Initial random direction (diagonal: NE/SE/SW/NW)
  dir = rand() % 4

  do:
    // Terrain promotion (raise terrain along river)
    switch terrain:
      DESERT: terrain = PLAINS; if rand()%10 > 1 or length > 3: terrain = GRASSLAND
      PLAINS: if rand()%10 > 3: terrain = GRASSLAND
      FOREST: if rand()%10 > 2: terrain = GRASSLAND
      HILLS/MOUNTAINS: terrain = GRASSLAND
      TUNDRA: terrain = PLAINS
      GLACIER: terrain = TUNDRA
      SWAMP: if rand()%10 > 3: terrain = JUNGLE

    // Mark river bit (0x80) on terrain byte
    terrain |= 0x80
    *tile_ptr = terrain
    river_length += 1

    // Check if any diagonal neighbor is ocean (river reached coast)
    reached_coast = false
    for d = 0 to 3:
      nx = wrap(x + RIVER_DX[d])
      ny = y + RIVER_DY[d]
      if is_tile_valid(nx, ny) and is_ocean(nx, ny):
        reached_coast = true

    // Meander: adjust direction slightly
    dir = (dir + rand()%2 - (river_length & 1)) & 3
    x = wrap(x + RIVER_DX[dir])
    y += RIVER_DY[dir]

    if is_tile_valid(x, y):
      terrain = tile(x, y).byte5  // scratch layer has terrain copy

  while !reached_coast and is_tile_valid(x, y) and !(terrain & 0x80)

  // Accept or reject river
  min_length = 5 - attempts / 800

      if valid and terrain == FOREST and near_equator:
        rand()  // placeholder (jungle conversion partially decompiled)
  else:
    // Too short — revert terrain
    copy_byte_layer(terrain_layer, scratch)

  progress_tick()
```

## Key Findings

### Game Logic Functions Worth Porting

1. **taxrate_recalc_totals** (0x0040c480) — Empire-wide gold/science/maintenance totals. Important for any tax rate adjustment UI. **TH priority**.

2. **check_and_show_tax_dialog** (0x0040decc) — Rate clamping after government change. The rate clamping logic exists in JS but the forced-check-on-change behavior should be verified. **TM priority**.

3. **grassland_has_shield** (0x0040bcb0) — The exact binary formula `(-(y + x>>1) - x) & 2 == 0` should be verified against the JS `calcTileShields` implementation. **TL priority** (likely already correct but untested).

### Fully Ported (P = 3)
- `generate_world_map` (FUN_00408d33) → `engine/mapgen.js:generateMap()`
- `mapgen_calc_fertility` (FUN_0040897f) → `engine/mapgen.js:calcFertility()`
- All continent/river placement subfunctions → inline in `engine/mapgen.js`

### Partially Ported (PA = 7)
- `is_tile_valid` → inline bounds checks throughout engine
- `map_fill_byte_layer`, `map_fill_rect_byte_layer`, `map_copy_byte_layer` → array operations in mapgen.js
- `grassland_has_shield` → formula in production.js
- `get_max_tax_rate` → GOVT_FACTOR table in defs.js
- `balance_tax_rates` → balanceRates in econai.js

### Referenced (R = 4)
- `handle_revolution` → reducer.js SET_GOVERNMENT action
- `place_continent`, `check_land_bounds`, `try_create_inland_sea` → referenced in mapgen.js comments

### Not Needed (NN = 2)

- `crt_static_init` family — C runtime
